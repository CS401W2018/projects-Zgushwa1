Year in School:

The form tag has an "action" attribute. What does it do?

The action attribute specifies where to send the form data when the form is submitted.

The form tag has a "method" attribute. What does it do?

The method attribute specifies the HTTP method (GET or POST) to use when sending form data.

What is the difference between a "POST" and a "GET"?

GET appends data to the URL, while POST sends data in the request body, making it more secure for sensitive information.
