Why is the viewport meta tag important?

It is important because it makes it where the page is able to be seen from different types of devices.

How can you adjust the CSS for printing?

You would copy and paste the css into another css and rename it for printing. You would attach it to your html file and make corrections only on the css file for printing

When would you want to adjust the CSS for printing?

You would want to adjust the css for printing when you want to make sure that the page is able to be printed out and look good on paper.
